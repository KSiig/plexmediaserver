# Plexmediaserver

Simple ansible role to install [Plex Media Server](https://www.plex.tv/media-server-downloads/)

## Role Variables

All variables can be seen in `defaults/main.yml`

## Example Playbook

Example of how to use role:

    - hosts: all
      roles:
         - plexmediaserver

## Dependencies

None
